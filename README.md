# 📚 Student Grade Calculator

A command-line Student Grade Calculator built with Python!

## 📌 About
Enter student name and marks for multiple subjects — get instant grades, GPA, performance analysis, and a detailed result card!

## 🚀 Features
- Supports multiple subjects
- Automatic Grade calculation (A+, A, B, C, D, F)
- GPA calculation (out of 4.0)
- Performance analysis (Outstanding, Excellent, Good, etc.)
- Highlights Best Subject & Subject that needs improvement
- Pass / Fail result
- Calculate for multiple students

## ▶️ How to Run

```bash
# Clone the repository
git clone https://github.com/abhitabchoudhary16-star/student-grade-calculator.git

# Navigate to the folder
cd student-grade-calculator

# Run the program
python student_grade_calculator.py
```

## 🎯 How to Use
1. Run the program
2. Enter the student's name
3. Enter subject names and marks one by one
4. Type `done` when all subjects are entered
5. View the detailed Result Card! 🎉
